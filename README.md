# 🏖️ Sunshade: AI paper analysis tool v0.1.1

### ✨ Key Features

- **OpenAI OAuth**: OpenAI 계정(ChatGPT Plus/Pro)을 안전하게 연동하여 앱 내에서 바로 AI 기능을 사용할 수 있습니다.
- **AI 분석**: PDF 논문의 키워드, TL;DR (3줄 요약), 그리고 심층 구조화된 요약을 자동으로 생성합니다.
- **문맥 인식 채팅**: 문서의 전체 내용을 기반으로 구체적인 질문과 답변이 가능한 채팅 기능을 제공합니다.
- **히스토리 & 즐겨찾기**: 읽은 논문 기록을 로컬에 자동 저장하고, 중요한 문서는 즐겨찾기로 관리할 수 있습니다.

### 🚀 Quick Start

[Download](https://github.com/blu-y/sunshade/releases/latest)

### 🛠️ Install Manually

#### Manual Install Requirements

- Node.js (v18 이상 권장)
- OpenAI 계정 (Codex 백엔드 사용 시 ChatGPT Plus/Pro 권장)

#### Manual Install Steps

```bash
git clone https://github.com/blu-y/sunshade.git
cd sunshade

# install dependencies
npm install

# run
npm start
```

### ⚠️ 주의사항

- 이 앱은 개인 연구 목적으로 ChatGPT Plus/Pro 구독과 함께 사용해야 합니다. 개인 연구 용도 외에는 OpenAI API를 사용하십시오.
- OpenAI Codex를 우회하여 사용하기 때문에, OpenAI의 정책에 따라 로그인이 불가능할 수 있습니다.
